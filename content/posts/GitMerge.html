Jak działa komenda git merge?

Komenda git merge służy do łączenia zmian z jednej gałęzi do drugiej w systemie kontroli wersji Git. Przeprowadzając operację scalania, Git analizuje historię obu gałęzi i tworzy nowy commit scalenia, integrując zmiany z obu gałęzi w jeden spójny stan projektu. Jeśli zmiany w tych gałęziach nie kolidują ze sobą, proces jest automatyczny; w przeciwnym razie użytkownik musi ręcznie rozwiązać konflikty.
